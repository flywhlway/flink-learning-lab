# L6 · 连接器 / CDC / 湖仓

> 共 24 题。先自答，再展开「参考答案」。链回 `docs/`/`ai/`/`projects/` 仅作延伸，不替代本题答案正文。

152. Flink CDC YAML Pipeline 相比手写 DataStream CDC 的价值？Schema Evolution？

**参考答案：** 答案：声明式源汇、转换链短，统一运维；Schema Evolution 按连接器策略（加列/宽表）配置并需下游兼容。手写 DS 灵活但成本高。见 docs/08-cdc、e08。

153. Paimon 与 Iceberg 在流式更新场景的根本差异？

**参考答案：** 答案：Paimon 面向流更新/主键表与 LSM 式更新友好；Iceberg 强项表格式生态与批分析，行级更新模型不同。选型看「流式变更频率」与湖仓工具链。见 docs/09-lakehouse。

154. Kafka Source 起始位点与组管理注意点？

**参考答案：** 答案：committed vs earliest/latest；组 id 冲突会抢消费。作业参数外置 group-id。见 docs/07-connectors、best-practice #7。

155. ClickHouse Sink 的幂等与批量？

**参考答案：** 答案：CH 常 at-least-once + 可重表/去重模型；批量与 flush 影响延迟。p03/p01 用 HTTP/JDBC SinkV2 模式按项目文档。见 e07、projects。

156. 连接器版本为何必须跟 Flink 版本矩阵走？

**参考答案：** 答案：API 与 shaded 依赖绑定；错版本是 NoClassDef/序列化地狱。SSOT：根 README + pom 属性。

157. Debezium 心跳与 WAL 膨胀？

**参考答案：** 答案：低流量表需心跳防 slot/位点停滞；忽略则 CDC 延迟飙升。见 docs/08-cdc。

158. 多表 CDC 合并拓扑注意？

**参考答案：** 答案：表过滤、路由、主键冲突、schema 漂移。YAML pipeline 路由规则要测。

159. Kafka 分区数与并行度关系？

**参考答案：** 答案：源并行度 ≤ 分区数才有意义；过多并行度空转。

160. 事务性 sink 与非事务性切换的数据风险？

**参考答案：** 答案：语义从 EOS 退到 at-least-once 会放大重复；需下游幂等并沟通业务。

161. 对象存储作为 sink 的小文件问题？

**参考答案：** 答案：高分区+短 checkpoint → 海量小文件。用分区合并、更长滚动、湖格式 compaction。见 docs/09-lakehouse。

162. Redis 作维表/特征的超时与降级？

**参考答案：** 答案：Lookup 超时走默认值/旁路；禁止同步客户端阻塞算子线程。见 p02 规划、best-practice #4。

163. Postgres logical decoding 前提？

**参考答案：** 答案：wal_level=logical 等；本仓库 PG 镜像已开。见 docker、docs/08-cdc。

164. 连接器指标哪些最有用？

**参考答案：** 答案：源 lag、checkpoint 对齐、错误计数、事务提交失败。结合值班五指标。

165. Schema Registry 何时引入？

**参考答案：** 答案：多消费者契约稳定时；学习工程可用 JSON+文档契约，避免未登记组件挡门禁。

166. FileSystem sink 与 checkpoint 的可见性？

**参考答案：** 答案：pending→finished 提交与 CP 对齐；读端需理解分区提交语义。

167. Kafka 事务性读-处理-写链路？

**参考答案：** 答案：源位移进 CP + 事务 sink commit on CP；超时参数对齐。

168. Iceberg 流写的提交周期？

**参考答案：** 答案：受 checkpoint 影响；过频小文件。

169. CDC 删除事件如何保留？

**参考答案：** 答案：下游 tombstone/软删字段；湖表按格式支持 delete。

170. 连接器报序列化错误优先查什么？

**参考答案：** 答案：类型推导、时区、嵌套 ROW、版本错。

171. 多环境 endpoint 如何注入？

172. 湖仓主键表与 append 表如何选？

**参考答案：** 答案：需要更新/删除用主键表；纯事实流用 append。

173. 连接器敏感配置如何防泄漏？

174. HTTP Sink 重试策略？

175. Paimon compact 与流写并发？
